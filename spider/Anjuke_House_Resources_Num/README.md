# 安居客各地房源数量爬虫文档

> 最新有效性检查时间：2020.12.28

**爬虫类型**：单次运行爬虫

**爬虫依赖第三方模块**：crawlertool、Selenium4R

**爬虫功能**：采集安居客各个城市的房源数量

**爬虫参数**：

| 参数名    | 参数功能                                                     |
| --------- | ------------------------------------------------------------ |
| city_code | 城市编码，可以通过安居客城市编码列表爬虫（Anjuke_City_Code_List）采集 |

**爬虫返回结果数据**：

| 字段名    | 字段内容     |
| --------- | ------------ |
| city_code | 城市编码     |
| city_num  | 城市房源数量 |
