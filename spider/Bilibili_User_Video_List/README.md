# B站UP主发布视频列表爬虫文档

> 最新有效性检查时间：2020.12.28

**爬虫类型**：单次运行爬虫

**爬虫依赖第三方模块**：crawlertool

**爬虫功能**：采集B站UP主发布的视频列表

**爬虫参数**：

| 参数名 | 参数功能                                            |
| ------ | --------------------------------------------------- |
| mid    | 目标UP主的账号ID（UP主主页Url中的形如20165629的数） |

**爬虫返回结果数据**：

| 字段名      | 字段内容     |
| ----------- | ------------ |
| video_title | 视频标题     |
| video_play  | 视频播放次数 |
