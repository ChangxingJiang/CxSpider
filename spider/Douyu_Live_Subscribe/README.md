# 斗鱼直播间订阅数爬虫文档

> 当前爬虫已失效！！！

**爬虫类型**：单次运行爬虫

**爬虫依赖第三方模块**：crawlertool、Selenium4R

**爬虫功能**：依据直播间Url列表，采集列表中直播间的订阅数

**爬虫参数**：

| 参数名   | 参数功能                                        |
| -------- | ----------------------------------------------- |
| driver   | Selenium4R.Chrome浏览器对象                     |
| live_url | 直播间地址（形如：https://www.douyu.com/96291） |

**爬虫返回结果数据**：

| 字段名         | 字段内容                                                     |
| -------------- | ------------------------------------------------------------ |
| live_url       | 直播间地址                                                   |
| text_subscribe | 直播间订阅数                                                 |
